Holiday Greetings - Landing Page Greeting Card

Holiday Greetings is an animated landing page greeting card for you and your team to wish your visitors, clients and business partners a Merry Christmas and a Happy New Year and show them how much you appreciate them in an inventive and ecological way.

Getting Started

Installation

Holiday Greetings - Landing Page Greeting Card is a HTML template so there's no really "installation" required just unzip the main file you've downloaded from themeforest and after unziping you have your all template files ready to be used..


File Structure

  • css folder: contains css stylesheet files. In these files it is defined how to display HTML elements.
  • fonts folder: contains font files for social icons.
  • images folder: contains all necessary images for website
  • media folder: contains audio files
  • js folder: contains javascript files. These files provide enhanced user experience and dynamic effects.

Change menu position

Menu can be aligned to left or right. All you need to do is to add the appropriate css class. The following example would pull the menu to the right.

Available classes are:

  • left: For pulling the menu to the left
  • right: For pulling the menu to the right
  • textleft: For aligning the text to the left
  • textright: For aligning the text to the right
  • uppercase: For capitalizing the letters
